# 原理

使用TCP或者UDP等协议向目标端口发送指定标志位等的数据包,等待目标返回数据包,以此来判断端口状态。

# 探测

1. 使用nmap探测。 nmap -A -v -T4 目标
2. 使用 massan 探测。
3. 使用在线网站探测http://tool.chinaz.com/port/
